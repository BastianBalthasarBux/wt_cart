What else is there?
^^^^^^^^^^^^^^^^^

There are additional extension available which extend wt\_cart and add
valuable functions.

wt_cart_order
"""""""""""""

The extension **wt_cart_order** adds a backend module and uses some hooks
of wt_cart. With the help of the backend module the TYPO3 editor/ admin
has access to an overview listing all orders which are placed till now.
Furthermore, using the list module allows one to show the order data.
That way one can get tons of information about the single order.

You can download the extension from our `github <https://github.com/extcode/wt_cart_order>`_ account.

|logo-1|

wt_cart_pdf
"""""""""""

The extension **wt_cart_pdf** adds tons of functions to render PDF
documents like invoices and delivery receipts. These documents can
be attatched automatically to the email confirmation. It is possible
to customize the documents extensively.

You can download the extension from our `github <https://github.com/extcode/wt_cart_pdf>`_ account.

|logo-2|

wt_cart_paypal
""""""""""""""

The extension **wt_cart_paypal** implements PayPal Standard as
payment method in wt_cart. There is no support for PayPal Express
so far.

Soon you will be able to download the extension from our github account.
