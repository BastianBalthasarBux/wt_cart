How can I use tt\_news together with wt\_cart?
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

If you want to connect tt\_news with wt\_cart we recommend using a
small extension called 'ttnews\_wtcart' This extension can be
downloaded from TER. It works together with tt\_news version 3.x.
